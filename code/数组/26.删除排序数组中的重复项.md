## 法一：遍历数组，与数组中的前一项比较，重复就删除该项（100ms）
### 删除数组方法 
    ```js
        nums.splice(i,1);
    ```
### 注：
每次删除后数组的长度会随之变化，直接i++会隔过一项，所以要先i--
### 代码
``` js
    var removeDuplicates = function(nums) {
    if(nums.length <= 1){
        return nums.length
    }
    for(i = 1; i < nums.length; i++){
        if(nums[i-1] == nums[i]){
            nums.splice(i,1);
            i--;
        }
    }
    return nums.length
};
```
